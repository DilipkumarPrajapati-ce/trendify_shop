# Trendify Frontend

This is a starter Next.js + TypeScript + Tailwind CSS frontend for Trendify.

Quick start:

```bash
cd frontend
npm install
npm run dev
```

Stack included: Next.js (App Router), TypeScript, Tailwind CSS, Framer Motion, GSAP, React Three Fiber, Drei, Lucide Icons, React Hook Form, Zod.

Notes: Add 3D assets into `public/models` and extend `components/Hero3D.tsx` to load your GLTF/GLB models.
